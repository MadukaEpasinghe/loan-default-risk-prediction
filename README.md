# 💳 Loan Default Risk Prediction

## 📌 Overview
This project builds a machine learning model to predict whether a customer is likely to default on a loan based on financial and demographic data.

---

## 🧠 Business Problem
Financial institutions face significant losses due to loan defaults.  
Accurately identifying high-risk customers before issuing loans is critical for minimizing financial risk.

---

## 💡 Solution
A predictive model was developed to classify customers into:
- Low risk
- High risk

This enables data-driven decision-making in credit approval processes.

---

## ⚙️ Tech Stack
- Python
- Pandas, NumPy
- Scikit-learn
- Matplotlib / Seaborn

---

## 📊 Key Features Used
- Credit score
- Income level
- Debt-to-income ratio
- Loan amount
- Employment status

---

## 📈 Model Performance
(Images)

Example:
- Accuracy: 87%
- Precision: 82%
- Recall: 79%

---

## 🔍 Key Insights
- Customers with high debt-to-income ratio are more likely to default
- Lower credit scores strongly correlate with higher risk
- Income stability plays a critical role
